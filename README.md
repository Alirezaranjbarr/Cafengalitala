18- {فـًٍʘًٍʘـنـًٍʘًٍʘـدََقـًٍʘًٍʘـ ٌۉفـًٍʘًٍʘـلـًٍʘًٍʘـفـًٍʘًٍʘـلـًٍʘًٍʘـ} 19- {فـؒؔـٓٓـؒؔ◌͜͡◌نـؒؔـٓٓـؒؔ◌͜͡◌دقـؒؔـٓٓـؒؔ◌͜͡◌ وفـؒؔـٓٓـؒؔ◌͜͡◌لـؒؔـٓٓـؒؔ◌͜͡◌فـؒؔـٓٓـؒؔ◌͜͡◌لـؒؔـٓٓـؒؔ◌͜͡◌} 20-
